üìñ Overview

Bitcraft Market Helper is a web-based tool that helps you browse and analyze the Bitcraft marketplace. It provides two main interfaces:

üéÆ Market Search

Search for individual items and view detailed market statistics, buy/sell orders, and pricing information.

‚öîÔ∏è Gear Finder

Browse and compare gear items by category, tier, and rarity with prices across all rarity levels in a table format.

üí° Tip: All market data is provided by bitjita.com and is updated in real-time when you search or refresh.

üéÆ Market Search Page

How to Use

  1. Search for Items: Start typing an item name in the search box. A dropdown will appear with matching items.
  2. Select an Item:
    • Click an item from the dropdown to view detailed market data
    • Press Enter to see multiple matching items in a table
  3. Filter by Tier: Use the tier dropdown to narrow your search to specific item tiers.
  4. Region Filter: By default, only Solvenar (Region 4) data is shown. Check "Show all regions" to include data from all regions.

Understanding the Data

Market Statistics

  • Lowest Sell: Best price to buy
  • Highest Buy: Best price to sell
  • Recent Sell Orders: Number of active sell listings
  • Recent Buy Orders: Number of active buy requests

Order Lists

  • Shows claim names (sellers/buyers)
  • Quantity available or wanted
  • Price per unit
  • Click üìã to copy prices

Multi-Item Search

Press Enter after typing a search term to see up to 20 matching items in a table format. This is great for comparing prices across similar items.

üí° Tip: Click on item names in search results to open them on bitjita.com for more details or to place orders.

‚öîÔ∏è Gear Finder Page

How to Use

  1. Select Filters: Choose one or more options from Category, Tier, and/or Rarity filters.
  2. Multiple Selection: Hold Ctrl (or Cmd on Mac) and click to select multiple options in each filter.
  3. View Results: Results appear automatically in a table showing prices for each rarity level.
  4. Sort Data: Click any column header to sort by that column (click again to reverse order).
  5. Filter Results: Use the search box in the results section to filter displayed items by name or category.

Available Categories

  • Blacksmith Tool, Carpenter Tool, Mason Tool, Tailor Tool
  • Farmer Tool, Fisher Tool, Forager Tool, Forester Tool, Hunter Tool, Miner Tool, Scholar Tool, Leatherworker Tool
  • Cloth Clothing, Leather Clothing, Metal Armor

Rarity Levels

Items can have different rarities, each with different stats and prices:

  • Common: Basic items (gray)
  • Uncommon: Improved items (green)
  • Rare: Superior items (blue)
  • Epic: Exceptional items (purple)
  • Legendary: Best-in-class items (orange)
üí° Tip: The table shows the lowest sell price for each rarity. "N/A" means there are currently no market listings for that rarity.

‚ùì Frequently Asked Questions

Q: Where does the market data come from?
All market data is provided by bitjita.com, which aggregates real-time market information from the Bitcraft game.
Q: How often is the data updated?
The data is fetched in real-time when you search or refresh. Market conditions change frequently, so use the Refresh button to get the latest information.
Q: Why do I see "N/A" for some prices?
"N/A" means there are currently no active market listings for that item or rarity. This could mean the item is not currently being sold or bought by anyone.
Q: Can I place orders through this tool?
No, this is a read-only tool for browsing and analyzing market data. To place orders, click on item names to visit bitjita.com or use the in-game marketplace.
Q: What's the difference between "Highest Buy" and "Lowest Sell"?
  • Lowest Sell: The cheapest price someone is willing to sell for. This is the best price if you want to buy the item.
  • Highest Buy: The highest price someone is willing to pay. This is the best price if you want to sell the item.
  • The difference between these is called the "spread" - it represents the market's buying and selling gap.
Q: Why is the app showing "Failed to fetch market data"?
This could happen due to:
  • Network connectivity issues
  • The bitjita.com API being temporarily unavailable
  • The item having no market data available
Try refreshing the page or checking your internet connection. If the problem persists, the API may be down temporarily.
Q: How do I select multiple filters in Gear Finder?
Hold down Ctrl (Windows/Linux) or Cmd (Mac) while clicking to select multiple options in the Category, Tier, or Rarity dropdowns.
Q: What does the claim name represent?
The claim name shows which player's claim (settlement) is selling or buying the item. This can help you identify specific sellers or find items from claims you prefer to trade with.

üí° Tips & Tricks

Finding the Best Deals
  1. Use Market Search to find specific items you need
  2. Check both "Lowest Sell" price and available quantities
  3. Compare prices across different rarities using the multi-item search
  4. Use Gear Finder to compare similar items (e.g., all Tier 3 tools)
Efficient Gear Shopping
  1. Go to Gear Finder
  2. Select your desired category (e.g., "Miner Tool")
  3. Select your tier range
  4. Compare prices across all rarities at once
  5. Click column headers to sort by price
  6. Click item links to purchase on bitjita.com
Market Analysis
  • Use the "Recent Orders" counts to gauge market activity
  • A large spread between Highest Buy and Lowest Sell indicates low market liquidity
  • Compare regional prices by toggling "Show all regions"
  • Check multiple tiers to find the best value for your budget
üí° Pro Tip: Bookmark both pages for quick access. Market Search is better for specific items, while Gear Finder excels at comparing similar items across rarities.

üîß Troubleshooting

Search Results Not Appearing
  • Make sure you've typed at least one character
  • Check if your tier filter is too restrictive
  • Try clearing the search and starting over
  • Refresh the page to reload the items database
Tooltips Not Showing
  • Make sure you're hovering over the blue "?" icons
  • On mobile, tooltips may not work - refer to this help page instead
  • Check if your browser allows CSS hover effects
Slow Loading Times
  • When filtering by region, loading is slower as it makes individual API calls
  • Use "Show all regions" for faster bulk loading
  • Try selecting fewer items/filters at once
  • Check your internet connection speed
Gear Finder Shows No Results
  • Make sure you've selected at least one filter (Category, Tier, or Rarity)
  • Try broadening your filters (select more options)
  • Check if there are actually items matching your criteria
  • Refresh the page to reload data
